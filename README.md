# Simple-Discord-Music-Bot
A Simple Music Bot Made In Discord.py supporting slash commands

## Setup
**Install Requirements**

```
pip install -r requirements.txt
```

**Edit Token**

###### replace 'YOUR TOKEN HERE' with your token
```
YOUR TOKEN HERE
```
```
NDk2MjY5NTI5MTA5MzE4NjYx.....
```


**Run Discord Bot**
```
python bot.py
```

## Commands


Music Commands
-  /play - Plays Song
- /skip - Skips Song
- /loop - Loops Song
- /pause - Pauses Song
- /resume - Resumes Song
- /queue - Displays Queue
- /clear - Clears Queue
- /leave - Leaves VC

Other Commands

- /meme - Random Meme
- /ping - Pings The Bot
